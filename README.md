## :star: Objetivo:

<p align='Justify'> 
Este trabalho teve como objetivo a construção de uma calculadora simples, que teve como ponto principal a o estudo e prática da linguagem Kotlin no desenvolvimento Android Nativo o a resolução de bugs encontrados em outras ferramentas similares, como:
<p/>

* A possibilidade de digitar expressões matemáticas repetidas vezes, como:

 ~~~ 
 +++ , --- , /// e *** 
 ~~~

* A possibilidade de digitar pontos/virgulas repetidas vezes. como:

 ~~~ 
 ... , 8.2.3.6 
 ~~~
 
 * A possibilidade de digitar pontos/virgulas no início e fim de cada operação. como:
 
 ~~~ 
 .8+5+6 , 82.3+.6 
 ~~~

## :green_book: Tarefas realizadas:
- [x] Criação de uma calculadora básica.
- [x] Impossibilitar digitação de expressões matemáticas seguidas vezes.
- [x] Impossibilitar a digitação de pontos/virgulas seguidas vezes.
- [x] Impossibilitar a escrita de pontos/virgulas no início e fim de cada operação.
- [x] Limitar operações em duas casas decimais.- 
- [x] Dependendo da saída, verificar se deve ter casas decimais.
- [x] Não permitir o app a girar com a tela.
- [x] Mudança no design da tela.
